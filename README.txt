WpLicense
=========

Installing
----------

 Copy the wpLicense.php and wpLicense directory into the wp-content/plugins
 directory in your WordPress installation.

 Activate the wpLicense plugin in the WordPress admin tool, and then go
 to Options -> Content License to select a license.
